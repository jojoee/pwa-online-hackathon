## Getting started
1. Install Node.js
2. Install global dependencies: `npm install -g yarn && firebase-tools && bower`
3. Install dependencies: `yarn && bower install`
4. Start dev: `firebase serve`

```
Please run `npm run commit` instead of git commit
```
